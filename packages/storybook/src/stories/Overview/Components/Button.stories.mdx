# SmartB Button


<Button style={{margin:"20px auto", width:"auto", display:"block", position:"relative"}}>Click me</Button>


## Props

| Name      | Type | Default Value | Required | Description|
| :----:    |:----:|  :----:       |   :----: |   :----:   |
|children|`React.ReactNode`||no|The react children props|
|onClick|`(event: React.ChangeEvent<{}>) => void`||no||
|disabled|`boolean`|`false`|no|If true the button will be disabled|
|hoverEffect|`boolean`|`true`|no|If false the hoverEffect will be disabled|
|className|`string`||no|Add the className to the button and override the default properties|
|style|`React.CSSProperties`||no|Add the style to the button and override the default properties|
