# SmartB Card


<Card style={{position:"relative", width:"500px", margin:"20px auto"}}>
    <Typography variant="body2" color="textSecondary" component="p">
      The SmartB Card
    </Typography>
</Card>


## Props

| Name      | Type | Default Value | Required | Description|
| :----:    |:----:|  :----:       |   :----: |   :----:   |
|header|`string`||no|The text that will be displayed in the header of the card|
|children|`React.ReactNode or React.ReactNode[]`||no|The content that will be displayed in the body of the card. If it's an array the body will be divided by the multiples contents.|
|dividerText|`string`||no|The text that will be disabled with the divider|
|dividerDirection|`"horizontal" or "vertical"`|`"horizontal"`|no|The direction of the divider|
|logo|`"default" or "document"`|`"default"`|no|Choose the logo in the background|
|footer|`React.ReactNode"`||no|The content that will be displayed in the footer|
|className|`string`||no|Add the className to the button and override the default properties|
|style|`React.CSSProperties`||no|Add the style to the button and override the default properties|
