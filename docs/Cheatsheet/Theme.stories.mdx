# Theme

Pour utiliser les composants de la librairie il faut définir le composant `ThemeContextProvider` comme parent du projet en lui donnant
un `theme` que vous pouvez redéfinir et récupérer ci-dessous:

<ThemeContextProvider theme={myTheme}>
  <ThemeOverview />
</ThemeContextProvider>
